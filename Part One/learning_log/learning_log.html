Learning Log

In practical 1 I made a barebones site using HTML. I had to make use of HTML elements to attempt to accurately recreate an example web page that was provided for me. I found that that was a great way to refresh my knowledge of web development and HTML syntax as I had not done this for a long time. I also used this time to begin my final website, creating a portal as my intro page that contained links for both tasks. I also learned that additional pages can be added through hrefs to additional html files I created a repository for my work to have source control and arranged for my site to be hosted on Github Pages. I also created this learning log page and decided on a style that incorporated collapsibles. I decided to use Visual Studio Code for my development as it has an integrated terminal, live server feature, and a highlighting style that I found intuitive.
